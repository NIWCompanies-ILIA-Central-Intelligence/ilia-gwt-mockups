✔ Copy invitation link — removed from these rows (NIWD-1882 AC#3): the ⋮ More-actions menu on the pre-send rows intentionally omits Copy invitation link. It — and Resend — now live only on Invited loved ones, after admin approval / invite sent, when the link is actually live. Why: Send only signs DocuSign and queues the batch for NIW admin review; it does not email the recipient. Leaving a copy-link here would let a giftor Send, then copy a not-yet-approved link and email the recipient directly, bypassing the NIW admin-approval gate. Removing it closes that hole.
